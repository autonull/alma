## What Genuine Autonomy Looks Like

### Not This (v5 fake autonomy)
```
Pick topic from list → Generate text → Store snippet → Done
```

### This (v6 real autonomy)
```
Identify knowledge gap → Formulate question → Pursue answer → 
Integrate learning → Identify next gap → (continue)
```

**The difference:** v5 selects from a menu. v6 **creates its own objectives** based on what it doesn't know.

## What Makes This Autonomous

### 1. Self-Directed Goal Formation

**Not:** "Pick from menu"

**But:** "What don't I know? Let me find out."

```python
# Gap-driven goal generation
def identify_gaps(knowledge):
    gaps = []
    
    # Type 1: Mentioned but unexplained
    for concept in knowledge.mentioned_concepts:
        if concept not in knowledge.understood_concepts:
            gaps.append(Gap(concept, type="undefined"))
    
    # Type 2: Partial understanding
    for concept in knowledge.understood_concepts:
        if knowledge[concept].confidence < 0.7:
            gaps.append(Gap(concept, type="uncertain"))
    
    # Type 3: Missing connections
    for c1, c2 in knowledge.concept_pairs:
        if not knowledge.has_connection(c1, c2):
            gaps.append(Gap((c1, c2), type="unconnected"))
    
    return gaps
```

**Autonomy:** The agent identifies what it needs to learn.

### 2. Knowledge Drives Behavior

**Not:** "Store and ignore"

**But:** "What I know determines what I do next"

```python
def select_next_goal(agent):
    gaps = agent.knowledge.identify_gaps()
    
    # Prioritize by:
    # 1. Centrality (how connected to other concepts)
    # 2. Tractability (can we answer this?)
    # 3. Uncertainty (low confidence = high priority)
    
    scored = []
    for gap in gaps:
        score = (
            gap.centrality * 0.4 +
            gap.tractability * 0.3 +
            (1 - gap.confidence) * 0.3
        )
        scored.append((gap, score))
    
    best_gap = max(scored, key=lambda x: x[1])[0]
    return Goal.from_gap(best_gap)
```

**Autonomy:** Knowledge state determines behavior.

### 3. Completion Has Criteria

**Not:** "Generated text = done"

**But:** "Artifact meets standards = done"

```python
COMPLETION_CRITERIA = {
    'explain': {
        'min_length': 200,
        'must_include': ['definition', 'mechanism', 'example'],
        'coherence_threshold': 0.6
    },
    'compare': {
        'min_length': 300,
        'must_include': ['similarities', 'differences', 'recommendation'],
        'structure_score': 0.7
    },
    'resolve': {
        'confidence': 0.8,
        'evidence_count': 3,
        'counterarguments_addressed': True
    }
}

def check_completion(goal, artifact):
    criteria = COMPLETION_CRITERIA[goal.type]
    
    checks = []
    if 'min_length' in criteria:
        checks.append(len(artifact) >= criteria['min_length'])
    if 'must_include' in criteria:
        for concept in criteria['must_include']:
            checks.append(concept in artifact.lower())
    
    return all(checks)
```

**Autonomy:** Agent knows when it's done.
